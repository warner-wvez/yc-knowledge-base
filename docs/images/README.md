# Screenshots

Drop screenshots of the vault here so they show up in the main README.

The main README already references these filenames, so if you save your images with these exact names, they appear automatically with no other edits:

- `graph-view.png` : the Obsidian graph view of the whole vault (the visual that sells it).
- `note-example.png` : a single note open, showing frontmatter, summary, and links.
- `series-page.png` : a series page listing its entries (for example, Lightcone Podcast).
- `search.png` : a full-text search across the vault returning multiple hits.

How to capture the graph view in Obsidian:

1. Open the `vault/` folder as a vault.
2. Open the graph view (the icon in the left ribbon, or the command palette: "Graph view: Open").
3. Let it settle, zoom to frame the whole graph, and take a screenshot.
4. Save it here as `graph-view.png`.
